The Promises Specification

A promise represents the eventual result of an asynchronous operation. More infomation about promises and the standard can be found here.

Other Promises Libraries

Kris Kowal's Qis a very well known promise library and is also the basis of Angular's $q. Bluebirdis performance focused and also has support for the node.js module loader system. RSVPis a lightweight library with plenty of features but no progression. ES6 Promise Polyfillis written to be compliant with the ES6 Promises standard.
